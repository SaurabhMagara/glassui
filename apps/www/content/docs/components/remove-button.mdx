## Features

The `RemoveButton` component includes the following features:

- Stylish design with hover effects
- Confirmation dialog to prevent accidental removals
- Loading state while the removal is in progress
- Success state after the item is removed
- Shake animation when clicked (before opening the dialog)
- Toast notification upon successful removal

## Customization

The `RemoveButton` component uses Tailwind CSS classes for styling. You can customize its appearance by modifying the classes in the component's source code.

## Dependencies

This component relies on the following dependencies:

- `@/components/ui/button`
- `@/components/ui/alert-dialog`
- `@/components/ui/use-toast`
- `lucide-react` for icons

Make sure these dependencies are installed and properly imported in your project.
